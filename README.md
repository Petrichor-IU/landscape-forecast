# 自然景观预测

一个面向风光摄影的静态预测网页。输入地点和时间后，页面会先搜索精确地点候选，再调用 Open-Meteo 的公开接口，结合本地天文近似算法，对朝霞、晚霞、火烧云、星空银河、流星雨和云海给出概率评分与判断依据。

## 使用

直接用浏览器打开 `index.html` 即可。

可输入城市、景区名称，或经纬度：

```text
黄山光明顶
牛背山
杭州
30.13,118.16
```

## 数据和限制

- 细粒度地点搜索来自 OpenStreetMap Photon/Nominatim。
- 城市兜底搜索、天气和海拔数据来自 Open-Meteo。
- 月相和流星雨窗口为浏览器本地近似计算。
- 云海强依赖局地地形、山谷水汽和逆温，页面提供的是摄影出行前的概率筛选，不替代临近时段卫星云图、雷达和现场观察。
